18.1.1. border :-

For specifying table we use border property.

Example :-

table, th, td
{
    border: 2px solid blue;
}

Coding Example :-

html file :-

        <link rel="stylesheet" href="assets/css/style_115.css">
    </head>
    <body>
        <table>
            <th>Name</th>
            <th>Roll</th>
            <th>Result</th>

                <td>Abhiram</td>

            </tr>

18.1.3. border-spacing :-

This property is used to set the distance between the borders of adjacent cells. This will work only when border-collapse is separated. We can set this property to horizontal spacing and vertical spacing in the form of cm, px.

Example :-

table
{
    border-collapse: collapse;
    border-spacing: 50px 10px; 
}

18.1.6. table-layout :-

This property is used to set table layout. We can set this property either auto (default) or fixed.

Example :-

table
{
    table-layout: fixed;
}

Coding Example :-

html file :-

<!DOCTYPE html>
<html>
    <head>
        <title>Abhiram</title>
        <link rel="stylesheet" href="assets/css/style_127.css">
    </head>
    <body>
        <table>
            <tr>
                <td width="10%">RahulKumarSharma</td>
                <td width="90%">LalpurRanchi</td>
            </tr>
        </table>
    </body>
</html>

css file :-

table
{
    border: 1px solid black;
    width: 100%;
    border-collapse: seperate;
    table-layout: auto;
}
td
{
    border: 1px solid black;
}
output
