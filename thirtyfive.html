35.1.  border properties in css :-

  • border-left
  • border-right
  • border-top
  • border-bottom

35.1.1.  border-left properties :-

  • border-left-style
  • border-left-width
  • border-left-color
  • border-left

35.1.1.1.  border-left-style :-

  • solid :-
        It specifies a solid border
  • dotted :-
        It specifies a dotted border
  • dashed :-
         It specifies a dashed border
  • double :-
       It specifies a double border 
  • hidden :-
         This is very similar to none value, except in border conflict resolution for table elements
  • none :-
        This is default which specifies no border.
Coding Example :-

html file :-

<!DOCTYPE html>
<html>
    <head>
        <title>Abhiram</title>
        <link rel="stylesheet" href="assets/css/style_258.css">
    </head>
    <body>
        <div>
            <h1>Abhiram B S</h1>
        </div>
        <div>
            <h1>Javalli</h1>
        </div>
    </body>
</html>

css file :-

div
{
    border-left-style: solid;
}
output
